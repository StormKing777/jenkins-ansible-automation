# Jenkins & Ansible: Automated CI/CD Pipeline

## Project Summary
This project demonstrates **Continuous Integration and Continuous Deployment (CI/CD)**. I used **Jenkins** to "watch" my code and **Ansible** to automatically send that code to a server whenever I made a change.

* **Languages Used:** Python Scripting, YAML.
* **Environments Used:** Ubuntu Linux, AWS.
* **Technologies Used:** Jenkins, Ansible, Git.
